## git提交规范 Commitizen
   Commitizen是一个撰写符合上面Commit Message标准的一款工具，可以帮助开发者提交符合规范的Commit Message。
> 安装

```iterm2
# 安装提交
npm install commitizen -g
# 安装规范
npm install -g cz-conventional-changelog
```

安装好后，使用`cz`，`git cz`或者`git-cz` 来取代`git commit`
